# 🍕 Служба доставки еды «Вкусный Код» v1.0

Десктопное приложение для автоматизации работы службы доставки еды. Клиенты оформляют заказы, администраторы управляют меню и статусами.

## Стек технологий
- Python 3.10+
- Tkinter / ttk (графический интерфейс)
- SQLite3 (база данных)
- re, time (валидация и защита)

## Как установить
1. Установите Python 3.10+
2. Скачайте проект: `git clone https://github.com/Maxwin512/food-delivery.git`
3. Перейдите в папку: `cd food-delivery`
4. Ничего дополнительно устанавливать не нужно (только стандартная библиотека)

## Как запустить
1. Откройте терминал в папке с проектом
2. Выполните: `python food_delivery.py`
3. Вход для администратора: `admin` / `Admin123!`
4. Или зарегистрируйтесь как новый клиент

## Возможности
- **Клиент:** просмотр меню, оформление заказа, история заказов
- **Администратор:** CRUD меню, управление статусами заказов, статистика
- **Безопасность:** блокировка входа на 30 сек, валидация пароля и email
