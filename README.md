# 🍕 FoodDelivery — Служба доставки еды «Вкусный Код»

Десктопное приложение для автоматизации работы службы доставки еды.  
Позволяет клиентам оформлять заказы, а администраторам — управлять меню и статусами заказов.

---

## 🛠 Стек технологий

- **Язык программирования:** Python 3.10+
- **Графический интерфейс:** Tkinter (ttk)
- **База данных:** SQLite3
- **Дополнительные библиотеки:** re (регулярные выражения), time (блокировка входа)

*Встроенные модули Python — ничего дополнительно устанавливать не требуется.*
